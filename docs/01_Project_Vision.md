# 01 – Project Vision

## Vision Statement

To become the most complete, private, and secure family heritage platform — a permanent digital home for family history that any extended family can own, control, and pass down through generations.

## Problem Statement

Existing genealogy platforms:
- Are public by default (data visible to strangers)
- Store family data on third-party servers
- Don't support complex family structures (adoptions, remarriages, step-families)
- Lack approval workflows for multi-branch families
- Don't model family branches with separate admin ownership
- Provide poor privacy controls for living people
- Don't support rich media with detailed metadata
- Can't generate professional-quality family books and reports

## Our Solution

A **private, self-hostable** family platform where:
- The family **owns all data**
- Each family branch has its **own administrator**
- All changes go through an **approval workflow**
- **Living members are protected** by default
- The **family tree is a graph**, not a rigid hierarchy
- **Rich media** is stored with full metadata
- **AI assists** with data entry, deduplication, and discovery
- Professional **PDF reports** can be generated at any time

## Long-Term Goals

1. Support families of 10 to 10,000+ members
2. Handle 100+ years of family history
3. Support multi-language interfaces
4. Generate a publishable family book on demand
5. Allow family historians to collaborate across branches
6. Integrate with external genealogy sources (Geni, Ancestry, FamilySearch) via import/export
7. Support DNA data linking in a future version

## Non-Goals (for now)

- Public social network features
- Integration with social media
- Public family tree discovery
- Third-party data sharing

## Success Criteria

| Metric | Target |
|--------|--------|
| Persons stored | 1,000+ |
| Media files | 10,000+ |
| Concurrent users | 100+ |
| System uptime | 99.9% |
| Data export | Full JSON/XML/GEDCOM export at any time |
| Response time | < 300ms for API calls |
